# Real-Time-Clock
Real Time Clock ⏱

🔘 Idea of project

Set and displays time on seven-segments by using multiplexing.

🔘 Hardware:

- LCD 2*16 
- Keypad 4*4
- Variable resistance << To control the lighting intensity
- Resistances
- Button 
-AVR
-Atmega32A .
-6 Seven-segments.
-External crystal.

🔘 Software:

Main Application Layer
HAL Layer (LCD, keypad, seven segment). 
MCAL Layer (DIO, TIMER).

proteus design
